# 计划目录说明（plans/）

本目录存放**各阶段的详细计划**。整个仓库的文档分工如下：

| 文件 / 位置 | 角色 |
|---|---|
| [README.md](../README.md) | 仓库入口，说明这是什么项目、从哪开始读 |
| [ROADMAP.md](../ROADMAP.md) | 产品大方向：最终要做什么、分几个阶段、当前在哪 |
| [CURRENT_STATUS.md](../CURRENT_STATUS.md) | 当前小方向：现在具体在做什么、下一步是什么 |
| [CHANGELOG.md](../CHANGELOG.md) | 阶段成果记录：每个阶段已经完成了什么 |
| `plans/`（本目录） | 各阶段的详细计划文档 |
| GitHub Projects 看板 | 实时任务板（Todo / In Progress / Review / Done） |

> 看板入口：[GitHub Projects 看板](https://github.com/users/reagan-nz/projects/1/views/1)
