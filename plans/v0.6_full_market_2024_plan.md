# v0.6 full_market_2024 全量提取方案

_状态：规划中（2026-06-23）_

## 背景

eval1000_v2（同 cohort 重跑，2026-06-22）与 independent eval1000（新 cohort 泛化验证，2026-06-23）均已完成：

| run | 样本 | ok | 非金融 proxy |
|---|---:|---:|---:|
| eval1000_v2 | 1020 | 947 | 10.33/11 |
| independent eval1000 | 1000 | 918 | 10.30/11 |

管道泛化良好（Δ −0.04，PASS）。下一步：对全部 ~5300 家 A 股公司提取 2024 年报基础字段，建立可入库的全市场数据集。

## 目标

1. 抽取全部 A 股 2024 年报 11 项（工业）/ 金融子 schema 字段
2. 输出至 `outputs/generalization/full_market_2024/`，不覆盖历史 eval run
3. SQLite 导入 `run_name=full_market_2024`
4. 与 eval1000_v2 / independent 对比 proxy 率，验证规模泛化

## 样本设计

当前 `sample_universe.py` 不支持全市场（始终分层抽样）。需新建 **`lab/make_full_market_yaml.py`**（约 30 行），从 CNINFO stockList API 取全部 A 股，复用 `board_of` / `is_financial`，输出 `lab/eval_companies_full_market_2024.yaml`（Git 提交，可复现 universe 快照）。

预期：**~5300 家**，`financial: true` ~150–200 家。

## 批次策略

为避免 5300 家中途崩溃丢失 JSON（`eval_generalize.py` 仅在全量完成时写 `eval_results.json`），按板块拆 **5 个独立 batch**，各自独立 subdir：

| Batch | Board | 预估家数 | 预估耗时 |
|---|---|---:|---:|
| 1 | `sse_main` | ~1700 | ~6–8 h |
| 2 | `star` | ~560 | ~2–3 h |
| 3 | `szse_main` | ~1400 | ~5–7 h |
| 4 | `chinext` | ~1300 | ~5–6 h |
| 5 | `bse` | ~300 | ~1–2 h |
| — | **合计** | **~5300** | **~20–26 h** |

各 batch 顺序执行（不并行，避免 CNINFO 限流）；使用 `nohup` overnight 运行。

## 输出目录结构

```
outputs/generalization/full_market_2024/

  star/

  bse/
  eval_results.json  ← 5 batch 合并后

```

## 网络与缓存

- **VPN 关闭**：independent run 曾出现 18 次 ChunkedEncodingError，均因 VPN 所致
- **PDF 预拷贝（推荐）**：从 eval1000 / v2 / independent 预拷贝 ~2000 重叠 PDF+meta，节省 2–3 h 与 ~25 GB 下载
- **throttle 1.5**：全量 run 保持不变

## 前置条件

- [ ] `git status` 干净
- [ ] `df -h` ≥ **80 GiB** 可用（~70 GB 新 PDF + ~10 GB buffer）
- [ ] 机器可 overnight 无人值守 20–26 h
- [ ] VPN 关闭

## 关键步骤（命令摘要）

```bash
# Step 1: 生成全市场 YAML
.venv/bin/python lab/make_full_market_yaml.py lab/eval_companies_full_market_2024.yaml

# Step 2: 拆 5 个 board batch YAML
.venv/bin/python - <<'PY'
import yaml
cs = yaml.safe_load(open("lab/eval_companies_full_market_2024.yaml"))["companies"]
for board in ["sse_main","star","szse_main","chinext","bse"]:
    batch = [c for c in cs if c["board"] == board]
    yaml.safe_dump({"companies": batch},
                   open(f"lab/batch_{board}_2024.yaml","w",encoding="utf-8"),
                   allow_unicode=True, sort_keys=False)
    print(f"  {board}: {len(batch)}")
PY

# Step 3: （推荐）预拷贝重叠 PDF
# 见 plans/full_market_2024_extraction_plan.md Step 3

# Step 4: 顺序跑 5 批次（overnight）
mkdir -p outputs/generalization/full_market_2024/{sse_main,star,szse_main,chinext,bse}
for board in sse_main star szse_main chinext bse; do
  nohup .venv/bin/python lab/eval_generalize.py \
    --companies lab/batch_${board}_2024.yaml \
    --out outputs/generalization/full_market_2024/${board} \
    --throttle 1.5 \
    > outputs/generalization/full_market_2024/batch_${board}.log 2>&1 &
  wait   # 等待当前 batch 完成再跑下一个
done

# Step 5: 合并结果 + SQLite 导入
.venv/bin/python lab/db_import.py \
  --eval-dir outputs/generalization/full_market_2024 \
  --run-name full_market_2024 --limit 0
```

完整命令见 [full_market_2024_extraction_plan.md](full_market_2024_extraction_plan.md)。
