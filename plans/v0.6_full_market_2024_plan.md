## 目标

1. 抽取全部 A 股 2024 年报 11 项（工业）/ 金融子 schema 字段
2. 输出至 `outputs/generalization/full_market_2024/`，不覆盖历史 eval run

4. 与 eval1000_v2 / independent 对比 proxy 率，验证规模泛化

## 样本设计

当前 `sample_universe.py` 不支持全市场（始终分层抽样）。需新建 **`lab/make_full_market_yaml.py`**（约 30 行），从 CNINFO stockList API 取全部 A 股，复用 `board_of` / `is_financial`，输出 `lab/eval_companies_full_market_2024.yaml`（Git 提交，可复现 universe 快照）。

预期：**~5300 家**，`financial: true` ~150–200 家。

## 批次策略（disk-safe 顺序）

为避免 5300 家中途崩溃丢失 JSON（`eval_generalize.py` 仅在全量完成时写 `eval_results.json`），按板块拆 **5 个独立 batch**，各自独立 subdir。

**默认按小→大顺序执行**（disk-safe）：`bse` → `star` → `szse_main` → `chinext` → `sse_main`。每 batch 完成后确认 `eval_results.json` / `eval_summary.md`，再删除该 batch 下的 PDF 与 `.cache`（保留 `company_profile.json`、`meta.json`）。

| Batch | Board | 预估家数 | 预估耗时 |
|---|---|---:|---:|
| 1 | `bse` | ~300 | ~1–2 h |
| 2 | `star` | ~560 | ~2–3 h |
| 3 | `szse_main` | ~1400 | ~5–7 h |
| 4 | `chinext` | ~1300 | ~5–6 h |
| 5 | `sse_main` | ~1700 | ~6–8 h |
| — | **合计** | **~5300** | **~20–26 h** |

各 batch 顺序执行（不并行，避免 CNINFO 限流）；使用 `nohup` overnight 运行。

> **重跑代价**：删除 PDF 后若需重跑某 batch，须从 CNINFO 重新下载（`meta.json` 仍保留，可加速定位）。

## 磁盘与 network（disk-safe 默认）

| 项 | 建议 |
|---|---|
| **磁盘** | 最低 **80 GiB**；若保留 PDF 建议 **100 GiB+**；disk-safe 模式下 **~88 GiB 可接受**，但每 batch 后须 `df -h` |
| **VPN** | **关闭** — independent run 曾出现 18 次 ChunkedEncodingError |
| **PDF 预拷贝** | **默认跳过** — 旧 eval run 的 PDF 已在 2026-06-23 预清理删除（~2812 份）；全量从 CNINFO 下载 |
| **batch 后清理** | 确认 `eval_results.json` 后，删除该 batch subdir 下 `*.pdf` 与 `.cache`；保留 profile / meta / JSON |
| **throttle** | **1.5**（全量 run 不建议低于 1.0） |

### 2026-06-23 预清理（已完成）

已从 eval1000 / eval1000_v2 / eval1000_independent_20260623 删除 **2812 个 PDF** 与 **3 个 `.cache`**（~10.5 GiB），保留全部 `eval_results.json`、`company_profile.json`、`meta.json`、`eval_summary.md` 及对比报告。可用空间 **84 → 95 GiB**。

## 关键步骤（命令摘要）

```bash
# Step 1: 生成全市场 YAML
.venv/bin/python lab/make_full_market_yaml.py lab/eval_companies_full_market_2024.yaml

# Step 2: 拆 5 个 board batch YAML
.venv/bin/python - <<'PY'
import yaml
cs = yaml.safe_load(open("lab/eval_companies_full_market_2024.yaml"))["companies"]
for board in ["sse_main","star","szse_main","chinext","bse"]:
    batch = [c for c in cs if c["board"] == board]
    yaml.safe_dump({"companies": batch},
                   open(f"lab/batch_{board}_2024.yaml","w",encoding="utf-8"),
                   allow_unicode=True, sort_keys=False)
    print(f"  {board}: {len(batch)}")
PY

# Step 3: 跳过 PDF 预拷贝（disk-safe 默认）

# Step 4: 顺序跑 5 批次 + batch 后清理（小→大）
mkdir -p outputs/generalization/full_market_2024/{bse,star,szse_main,chinext,sse_main}
for board in bse star szse_main chinext sse_main; do
  .venv/bin/python lab/eval_generalize.py \
    --companies lab/batch_${board}_2024.yaml \
    --out outputs/generalization/full_market_2024/${board} \
    --throttle 1.5 \
    2>&1 | tee outputs/generalization/full_market_2024/batch_${board}.log
  # 确认 checkpoint 后再删大文件
  test -f outputs/generalization/full_market_2024/${board}/eval_results.json
  find outputs/generalization/full_market_2024/${board} -name "*.pdf" -type f -delete
  find outputs/generalization/full_market_2024/${board} -type d -name ".cache" -prune -exec rm -rf {} +
  df -h .
done

# Step 5: 合并结果 + SQLite 导入
.venv/bin/python lab/db_import.py \
  --eval-dir outputs/generalization/full_market_2024 \
  --run-name full_market_2024 --limit 0
```

完整命令见 [full_market_2024_extraction_plan.md](full_market_2024_extraction_plan.md)。
